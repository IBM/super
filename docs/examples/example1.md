# Super Example 1: Simple Scaling via `-p`

<img src="images/runvis1.png" align="left" height="150">

Using the -p option, Super can run a fixed number of UNIX command
lines, in the Cloud.

```sh
super run -p3 -- printenv JOB_INDEX
[Job 1] 1
[Job 2] 2
[Job 3] 3
```
